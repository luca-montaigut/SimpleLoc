## Installation

`git clone git@gitlab.com:simpleloc/front_simpleloc.git`

`cd front_simple`

`npm install`

`echo REACT_APP_API_URL='http://localhost:8080' > .env`

`npm start`

Your website is now running on http://localhost:3000
